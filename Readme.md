
## Descripción

Este proyecto contiene la estructura base para el desarrollo de una aplicación web. Está organizado siguiendo una separación de responsabilidades entre:

- **HTML**: estructura de la página.
- **CSS**: estilos y diseño visual.
- **JavaScript**: lógica y comportamiento dinámico.
- **SRC**: recursos adicionales como imágenes, componentes, datos o archivos de apoyo.

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)

## Carpetas

### `/css`
Contiene las hojas de estilo del proyecto.

**Archivo principal:**
- `style.css`

### `/scripts`
Contiene los archivos JavaScript responsables de la lógica de la aplicación.

### `/src`
Contiene recursos del proyecto como:
- Imágenes
- Iconos
- Archivos JSON
- Componentes
- Recursos multimedia
